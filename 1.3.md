# Linux
## Linux的优良特性
	- 分时的多用户,多任务操作系统
	- 多数网络协议支持,方便的远程管理
	- 强大的内存管理和文件系统管理
	- 大量的可用的软件和免费的软件
	- 优良的稳定性和安全性
	- 良好的可移植性和灵活性
	- 可供选择的厂商多
## Linux常见的发行版本

# Linux命令

### 重启命令

		- reboot

		- shutdow -r now 立即重启(root用户使用)

		- shutdow -r 10 过10分钟自动重启(root用户使用)

		- shutdow -r 20:35 在时间为20:35时候重启(root用户使用)
	
		- 如果是通过shutdown命令设置重启的话,可以用shutdown -c 命令取消重启

### 关机命令

		- halt 立即关机

		- poweroff 立即关机

		- shutdow -h now 立即关机(root用户使用)

		- shutdow -h 10 十分钟后自动关机
